## Requirements

- Android 6.0 (API 23) or higher
- ~50 MB free storage
- Internet connection for balance/history sync

## Install the APK

1. Download `Octra-Wallet-v1.0.2-android.apk` from [GitHub Releases](https://github.com/Xyntera/Octra-Wallet/releases)
2. On Android: **Settings → Apps → Special app access → Install unknown apps** — enable for your browser or Files app
3. Open the APK file and tap **Install**

## Create a New Wallet

1. Launch Octra Wallet
2. Tap **Create Wallet**
3. Write down your **12-word seed phrase** and store it offline — this is the only way to recover your wallet
4. Set a **PIN** (4 digits) for local access
5. Optionally enable biometric unlock

## Import an Existing Wallet

Tap **Import Wallet** on the setup screen, then choose:

- **Seed phrase** — enter your 12 BIP-39 words
- **Private key** — paste your base64-encoded Ed25519 private key
